要推薦什麼？: The Apriori Algorithm

The downward closure property of frequent patterns: Any subset of a frequent itemset must be frequent
If {beer, diaper, nuts} is frequent, so is {beer, diaper}
i.e., every transaction having {beer, diaper, nuts} also contains {beer, diaper}

Scalable mining methods: Three major approaches

  1. 🍎 Apriori
  2. Freq. pattern growth
  3. Vertical data format approach

Apriori pruning principle: If there is any itemset which is infrequent, its superset should not be generated/tested!

  1. Set a support = #
  2. Count # for every single item in the DB [C1] -> Rule out the item < support [L1]
  3. Based on [L1], generate [C2] for two item pairs -> Rule out the item < support [L2]
  4. Repeat
