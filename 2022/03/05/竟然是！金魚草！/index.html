竟然是！金魚草！

自2021夏，NFT的各種訊息開始轟炸，近日看到生成作品的影片，想說來沾沾浪潮的泡沫(._.)。思來想去，有什麼適合本畫渣來嘗試的動物作品，鬼畜的金魚草便出現在腦海，於是，歡迎我們的金魚草醬。

實作 Code

  1. 創建資料夾
    先創 “metadata”, “layers”, “images”三個資料夾
    在 layers 資料夾中，再以 0X-ConponentName各式的資料夾儲存金魚草的各個 components。建議根據圖層疊加順序來作數字的順序，這樣在合成的時候就不用再檢視順序；且每張 png 要相同大小。

  2. Import 套件

  3. 對每個 Component 設置命名、權重、路徑
    權重會決定生成的機率，也就是稀有度。
    有幾個 Component （如金魚草就有，背景、魚的輪廓、魚鰭、眼睛等等）就會有幾組以下的代碼。

  4. 定義Functions：根據權重生成組合、確認生成是否為唯一

  5. 執行生成組合

  6. 設置 Token ID，編號

  7. 生成 METADATA_FILE

  8. 根據生成的組合，實際執行合成

  9. 最後！竟然是！金魚草！
